# tabs

简化 TabLayout 使用，使用 TabItemView 代替 TabLayout.TabView，支持远程动态加载图标

## Gradle

``` groovy
repositories {
    maven { url "https://gitee.com/ezy/repo/raw/cosmo/"}
}
dependencies {
    implementation "me.reezy.cosmo:tabs:0.7.0"
}
```
